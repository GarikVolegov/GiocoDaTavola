# Kit marketing SCHIERATI → NorthStar

Materiale operativo per usare **SCHIERATI** (party game gratuito) come canale di acquisizione
verso **NorthStar** (SaaS di orientamento e crescita professionale, Pro 24€/mese).

**Strategia in una riga:** *Allenati a decidere per gioco. Poi fallo sul serio.*
**Motore:** organico, 0€. **Canali:** YouTube (long + Shorts) + Instagram (feed + stories).
**Obiettivo:** far conoscere il brand **e** accumulare iscrizioni a NorthStar.

## Da dove parto domani (Fase 0, settimana 1)

1. Apri/rifinisci il canale **YouTube** e il profilo **Instagram** (bio con CTA + link in bio).
2. Prepara i **link tracciati** (vedi `tracking-e-link.md`).
3. **Gira il primo batch**: 1 sessione di gioco con amici → 1 episodio + 5–8 verticali.
4. Gira i **3 video hero** (`script-video-hero.md`).
5. Pubblica seguendo il **calendario** (`calendario-editoriale.md`).

## Cosa serve da te (autorizzazioni — il "5%")

- ✅ Confermare il **dominio di produzione** di NorthStar.
- ✅ Creare/darmi il **codice affiliato** per SCHIERATI (per tracciare le iscrizioni).
- ✅ Decidere se/quando **deployare** la build di SCHIERATI col ponte (categoria "Carriera" + CTA).
- ✅ Aprire gli **account social** (o darmi accesso) per la pubblicazione.
